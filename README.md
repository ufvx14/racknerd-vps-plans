## FAQ

**Q：免备案服务器是合法的吗？**
A：合法。免备案只是说服务器不在大陆境内，不需要走 ICP 备案流程。但你的网站内容仍要遵守服务器所在国家的法律，以及你受众所在地的法律。如果网站面向中国大陆用户且属于经营性网站，仍有一些合规要求需要自行了解。

**Q：RackNerd 支持哪些支付方式？**

**Q：能装 Windows 吗？**

**Q：能退款吗？**

**Q：能不能用来挂代理？**
